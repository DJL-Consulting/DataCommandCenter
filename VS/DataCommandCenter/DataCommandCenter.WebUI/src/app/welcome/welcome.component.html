Welcome to the Data Command Center!

The Data Command Center (DCC) combines database, integration, lineage, and metdata about any data environment into a single searchable resource.

There are two display views available: Metadata, which displays the item you select in the environment of it's surroundings, and Lineage, which shows both upstream and downstream objects in the lineage (data population) of the chosen object.

You can search for any server, database, object (table, view, stored procedure, fuction, trigger, etc.), column, or integration. Drop down the Search Options menu next to the search box to refine search criteria and choose Metadata or Lineage search.

Navigating the search canvas is easy and fun:
  • Zoom in and out using the scroll wheel on your mouse.
  • Hover over an item for a tooltip with some additional info about it.
  • Click on an item to highlight its relationships.
  • Then right-click for a detailed popup including all metadata (which can be edited by select users).
  • Double-click on an item to re-populate the graph with that item as the selected item/center.

Database metadata and lineage data is automatically populated by our custom crawler engines, which automatically populate all of the base information. Users from IT and Data Governance can then add/edit the user maintained metdata sections, choosing either from a pre-defined template of properties, or even adding new properties for specific objects as needed.
